# 🚀 Curso de Programación Orientada a Objetos (Java)

Bienvenido a este repositorio de apuntes y ejercicios de **Programación Orientada a Objetos (POO)**. Este proyecto está diseñado como una guía progresiva, desde los fundamentos de la sintaxis en Java hasta conceptos avanzados como polimorfismo, manejo de excepciones y persistencia de datos.

## 📌 Estructura del Repositorio

El contenido está organizado de forma temática para facilitar el aprendizaje secuencial:

- **[01-fundamentos](./01-fundamentos):** Introducción a métodos, paso de parámetros, miembros estáticos y lógica algorítmica básica.
- **[02-objetos-y-encapsulamiento](./02-objetos-y-encapsulamiento):** Definición de clases, instanciación, modificadores de acceso (`public`, `private`, `protected`) y sobrecarga de constructores.
- **[03-herencia-y-polimorfismo](./03-herencia-y-polimorfismo):** Reutilización de código mediante herencia, sobreescritura de métodos (`@Override`) y flexibilidad dinámica con polimorfismo.
- **[04-abstraccion](./04-abstraccion):** Implementación de clases abstractas y métodos abstractos para modelado complejo.
- **[05-excepciones-y-archivos](./05-excepciones-y-archivos):** Robustez de software mediante el manejo de errores (`try-catch`) y persistencia de información en archivos de texto.
- **[06-ejercicios-complementarios](./06-ejercicios-complementarios):** Casos de estudio prácticos que integran múltiples conceptos (Gestión de atletas, vehículos, geometría, etc.).

## 🛠️ Cómo ejecutar los proyectos

Cada carpeta es independiente y agnóstica de cualquier IDE (NetBeans, IntelliJ, VS Code). Para ejecutar cualquier ejercicio:

1. Navega a la carpeta del ejercicio:
   ```bash
   cd 03-herencia-y-polimorfismo/polimorfismo-spiderverse
   ```
2. Compila todos los archivos `.java`:
   ```bash
   javac *.java
   ```
3. Ejecuta la clase principal:
   ```bash
   java Main
   ```
